# VS Code 安装 Codex 插件教程：下载、配置与使用

本教程面向希望在 VS Code 中使用 OpenAI Codex 插件的开发者，包含基础介绍、插件安装步骤、Key 配置与使用方法。

---

## 01. VS Code 是什么？

VS Code（Visual Studio Code）是 Microsoft 推出的免费代码编辑器，你可以把它理解为一个专门用于写代码和管理项目的工作台。

它本身不局限于某一种语言，而是通过插件支持多类开发场景：

- 前端开发（HTML / CSS / JavaScript）  
- 后端开发（Python / Node.js / Go 等）  
- 脚本与自动化  
- Web、桌面应用等多类项目  

VS Code 的特点是 **轻量、灵活、扩展性强**，非常适合和 AI 编程助手（如 Codex）搭配使用。

---

## 02. Codex 是什么？

Codex 是面向编程场景的 AI 模型，相比普通聊天助手，它更擅长：

- 理解代码结构和上下文  
- 根据需求生成完整代码  
- 修改、补全、重构已有代码  
- 解释代码逻辑与报错原因  

在 VS Code 中使用 Codex 后，你可以直接在编辑器里：

- 用自然语言描述功能需求  
- 让 AI 生成或补全代码  
- 基于当前项目做定向修改  

可以把它理解为一个随时可用的 **编程助手**，帮助你更快把想法变成可运行的代码。

---

## 03. VS Code 安装 Codex 插件

1. 打开 VS Code。  
2. 进入左侧扩展市场（Extensions）。  
3. 在扩展商店搜索：

Codex – OpenAI’s coding agent

然后点击安装。  
4. 安装完成后，右侧活动栏会出现 Codex 图标。  

> 如果此时还无法正常使用，通常是因为还没完成后续的环境配置。

---

## 04. Codex 接入 VS Code（Key 与配置）

### 4.1 获取可用 Key

安装后必须配置 Key，插件才能正常工作。国内用户建议使用中转方式获取 Key，以提高稳定性：

[Codex 中转 Key 注册地址](https://codex.dakeai.cc/register?promo=DKAI)

### 4.2 中转站创建密钥并覆盖配置

1. 注册并登录后，在 **API 密钥页面** 创建 Codex 分组密钥。  
2. 在 **使用密钥** 页面，使用“覆盖配置脚本”方法覆盖 Codex 配置。  
3. 完成覆盖后，即可开始在 VS Code 中使用 Codex 插件。

---

## 05. 使用 Codex 插件

配置完成后，你可以在 VS Code 中：

- 直接在代码文件中输入自然语言描述，让 Codex 生成或补全代码  
- 使用快捷键快速触发代码生成（具体快捷键可在插件设置中查看）  
- 在当前项目上下文中修改或优化已有代码
